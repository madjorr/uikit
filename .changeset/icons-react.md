Add `@acronis-platform/icons-react` — React icon components generated from
`@acronis-platform/design-assets`. Ships the `stroke-mono` pack (40 icons) via
the `./stroke-mono` subpath export as tree-shakeable per-icon named exports
plus an `icons` registry. Icons use `currentColor` and bake the design-assets
scale + stroke rules into a `size` prop (1.6px stroke at 16, 2px at 24, 2.5px
at 32).
