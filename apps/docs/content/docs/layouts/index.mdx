---
title: Layouts
description: Page scaffolds and layout primitives for arranging an app's structure.
---

**Layouts** are the structural building blocks of a screen — full-page scaffolds
with named slots for other components, and low-level primitives for arranging
content.

- [App Shell](/layouts/app-shell) — the full-page application scaffold (sidebar +
  header + main slots).
- [Stack](/layouts/stack) — one-dimensional flex layout.
- [Grid](/layouts/grid) — responsive two-dimensional grid.
- [Section](/layouts/section) — a titled content block.

For the recommended way to assemble a full screen, see the
[App Shell pattern](/patterns/app-shell).
