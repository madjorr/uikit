## Usage

```tsx
import { Stack } from '@acronis-platform/ui-react';
```

`Stack` is a flexbox layout primitive: arrange children in a row or column with a
chosen `gap`, `align`, `justify`, and `wrap` — without hand-writing flex utilities.

> Design-pending v1, ported 1:1 from the legacy `shadcn-uikit` stack.

## Examples

<DemoReact>
  <StackDemo />
</DemoReact>

```tsx
<Stack direction="horizontal" gap="sm" align="center">
  <Button>Save</Button>
  <Button variant="ghost">Cancel</Button>
</Stack>
```

For two-dimensional layouts use [`Grid`](/layouts/grid).
