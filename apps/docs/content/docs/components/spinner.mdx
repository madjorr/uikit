title: Spinner
description: An internal spinning-ring primitive — use Loading for a standalone loading indicator.
---

> **Internal only, as of `ui-react` 0.58.0.** `Spinner` is no longer exported
> from `@acronis-platform/ui-react`'s public entry point. It remains an
> internal primitive consumed by [`Loading`](/components/loading) (its icon)
> and `Toast` (a small inline icon).
>
> Building a standalone loading state? Use [`Loading`](/components/loading)
> instead — it wraps the same spinning ring with a label and
> placement-context container tokens (`inline`, `onSurfacePrimary`,
> `onSurfaceSecondary`, `onScreen`).

## Migrating

```diff
- import { Spinner } from '@acronis-platform/ui-react';
- <Spinner size="lg" />
+ import { Loading } from '@acronis-platform/ui-react';
+ <Loading />
```

`Loading`'s three larger sizes (16 / 32 / 48px) line up with `Spinner`'s
`sm` / `lg` / `xl`; pass `hasLabel={false}` to drop the visible label while
keeping it announced via `aria-label`.
