## Usage

```tsx
import { Label } from '@acronis-platform/ui-react';
```

`Label` renders a native `<label>` with small, medium-weight type. Associate it
with a control via `htmlFor` (the control's `id`) so clicking the label focuses
the control and assistive tech reads it as the control's accessible name. When a
preceding `peer`-marked control is disabled, the label dims. This is a
design-pending v1; a `--ui-label-*` tier is pending a Figma pass.
