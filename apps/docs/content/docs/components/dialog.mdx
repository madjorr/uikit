## Usage

```tsx
import { Dialog } from '@acronis-platform/ui-react';
```

`Dialog` is the sanctioned way to build a dialog. A single `variant` prop
selects one of seven canned use-cases тАФ `default`, `rename`, `save changes`,
`reset password`, `discard changes`, `accept`, `read-only` тАФ each with its own
title, body copy, and footer buttons, so product code doesn't hand-assemble the
same header/body/footer boilerplate every time. `children` overrides the body
slot; `hasLoading` shows a spinner overlay across the body + footer. It's built
internally on Base UI's Dialog primitive (focus trap, scroll lock,
`Esc`/outside-press dismissal, and ARIA come from Base UI), but that primitive
is not itself exported тАФ if none of the seven variants fit your use case, raise
it with the design system team rather than reaching for a lower-level API.

An eighth `wide` variant is a legacy escape hatch тАФ no canned preset, a
fully custom `footer` тАФ kept only for backward compatibility with pre-existing
wider dialogs; prefer one of the seven canned variants for new work.

## Examples

<DemoReact>
  <DialogDemo />
</DemoReact>

A canned use-case, controlled open state:

```tsx
const [open, setOpen] = useState(false);

<Dialog variant="discard changes" open={open} onOpenChange={setOpen} />;
```

Override the body slot while keeping the header/footer chrome:

```tsx
<Dialog variant="default" defaultOpen>
  <p>Any custom body content.</p>
</Dialog>
```

Busy state:

```tsx
<Dialog variant="save changes" hasLoading open />
```

Localize the canned copy тАФ `title`, `secondaryLabel`, `primaryLabel`, and
`closeLabel` all override the variant's default English string:

```tsx
<Dialog
  variant="discard changes"
  title="Modifications non enregistr├йes"
  secondaryLabel="Retour"
  primaryLabel="Confirmer"
  closeLabel="Fermer"
  open
/>
```

When the dialog renders inside an isolated container (e.g. a shadow root), pass a
`portalContainer` so the popup inherits that scope's styles:

```tsx
<Dialog portalContainer={mountNode}>{/* тАж */}</Dialog>
```

Legacy `wide` variant тАФ a free-form footer for pre-existing wider dialogs that
don't fit one of the seven canned use-cases:

```tsx
import { Button, DialogClose } from '@acronis-platform/ui-react';

<Dialog
  variant="wide"
  size="large"
  title="Configure discovery agent"
  open
  footer={
    <>
      <DialogClose render={<Button variant="ghost">Cancel</Button>} />
      <Button>Configure</Button>
    </>
  }
>
  <p>Any custom body content.</p>
</Dialog>
```
